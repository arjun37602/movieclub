About the App!

Movie Club is a web app where all the most important facts about a movie are in one place! You get the poster, ratings, and more! All this gets delivered to your browser using the OMDb API. You no longer need to browse google for hours to find information about a movie. Simply type in a movie name and all the information will be delivered to your finger tips. Future functionality that we hope to add includes the streaming platform that the movie is available on.

Tech

  • HTML and CSS
  • JavaScript
  • OMDb API (The Open Movie Database)
